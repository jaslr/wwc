Junipa

Junipa is a robust educational platform tailored for the Australian education system, leveraging Google Cloud services to enhance data security and scalability.

The platform integrates seamlessly with existing educational tools and provides a unified experience through Single Sign-On, ensuring a secure and streamlined access for all users.

Product

Educational Platform

Sector

Product Goals

  • Enhance educational access through cloud-based solutions
  • Implement robust security measures with Google Cloud
  • Provide a seamless user experience with Single Sign-On

Product Technology

Developed using Google Cloud Platform, SAML for SSO

The Process

  • Integration with Google Cloud services
  • Design of user-friendly interfaces for educational stakeholders
  • Comprehensive testing and deployment
