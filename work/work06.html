Brontiq Books

A local-first invoice ledger system for managing client invoices, statements, and payments.

Overview

Brontiq Books was built to provide a simple, privacy-focused solution for freelancers and small businesses to manage their invoicing workflow. The application runs locally, ensuring your financial data stays on your machine.

Key Features

  • Client Statements - Generate comprehensive statements showing all invoices, payments, and outstanding balances per client
  • Invoice Management - Create, edit, and track invoices with line items, GST calculations, and payment status
  • Payment Tracking - Record payments against invoices with support for partial payments and credits
  • PDF Export - Download invoices and statements as professional PDF documents
  • Date Filtering - Filter statements by date range for specific reporting periods
  • Local-First - All data stored locally for privacy and offline access

Screenshots

Technology Stack

  • SvelteKit for the frontend framework
  • Local storage for data persistence
  • Tailwind CSS with DaisyUI for styling
  • PDF generation for document export
